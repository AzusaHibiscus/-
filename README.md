# 使用教程
1. 环境准备
首先，确保你已经安装了以下依赖库：

selenium
threading

2. 下载浏览器驱动
由于脚本使用的是Microsoft Edge浏览器，你需要下载对应的Edge浏览器驱动（Microsoft Edge WebDriver）

下载完成后，将驱动程序放在你的系统路径中，或者在脚本中指定驱动程序的路径（C盘）

4. 输入指令
